# GitHub Pages 上传包

这个目录已经是可直接发布的静态网站结构。

包含：
- `index.html`
- `assets/`
- `吴蕙琳_简历_2026-07-01.pdf`
- `.nojekyll`

发布方式：
1. 新建一个 GitHub 仓库（仓库名不要带 v2）
2. 把这个目录里的文件全部上传到仓库根目录
3. 在仓库 `Settings -> Pages` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. 保存后等待 1-3 分钟

建议仓库名示例：
- `lynn-portfolio-20260701`
- `wu-huilin-portfolio`
- `lynn-content-portfolio`
